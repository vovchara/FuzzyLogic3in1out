# Tooltip функціональність для графіків приналежності

## Що було додано

1. **Інтерактивні tooltip**: При наведенні курсора на будь-який з графіків функцій приналежності тепер відображається tooltip з детальною інформацією.

2. **Інформація в tooltip**:
   - **X-координата**: Точне значення по горизонтальній осі (0-100)
   - **Функції приналежності**: Список всіх лінгвістичних термів з їх значеннями в даній точці
   - **Кольорові індикатори**: Кожен терм має свій колір, що відповідає кольору на графіку
   - **Процентне відображення**: Значення приналежності відображається як у десятковому вигляді (0.000-1.000), так і у відсотках (0%-100%)

3. **Сортування**: Терми в tooltip сортуються за значенням приналежності (від найвищого до найнижчого).

4. **Адаптивне позиціонування**: Tooltip автоматично коригує своє положення, щоб не виходити за межі екрана.

## Технічна реалізація

### HTML зміни
- Додано div елементи з класом `graph-tooltip` для кожного графіка
- Кожен tooltip має унікальний ID: `csTooltip`, `rtTooltip`, `ecTooltip`, `srTooltip`

### CSS стилі
- Tooltip має темний напівпрозорий фон з blur ефектом
- Плавна анімація появи/зникнення
- Стрілка, що вказує на позицію курсора
- Адаптивний дизайн для різних розмірів екрана

### JavaScript функціональність
- `setupCanvasTooltips()`: Налаштовує обробники подій для всіх canvas елементів
- `handleCanvasMouseMove()`: Обробляє рух курсора над графіками
- `getMembershipInfoAtX()`: Знаходить значення функцій приналежності в конкретній точці
- `showTooltip()`: Відображає tooltip з відповідною інформацією
- `hideTooltip()`: Приховує tooltip при виході курсора з графіка
- `positionTooltip()`: Коригує позицію tooltip у межах видимої області

## Як користуватися

1. Наведіть курсор миші на будь-який з чотирьох графіків функцій приналежності
2. Tooltip автоматично з'явиться та показуватиме:
   - Поточну X-координату (значення від 0 до 100)
   - Значення всіх функцій приналежності в цій точці
3. Tooltip автоматично зникне, коли курсор покине область графіка

## Корисність

- **Точне читання значень**: Дозволяє точно визначити значення функцій приналежності в будь-якій точці
- **Порівняння термів**: Легко порівняти значення різних лінгвістичних термів в одній точці
- **Інтерактивність**: Робить графіки більш інформативними та зручними для аналізу
- **Освітня цінність**: Допомагає краще зрозуміти роботу нечітких множин та функцій приналежності
